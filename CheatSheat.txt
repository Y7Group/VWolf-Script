INIT - Initializes the script

STRING - Types a string
Example - STRING Hello, World!

DELAY - A delay in milliseconds
Example - DELAY 1000

START - Starts specified app
Example - START "cmd"

DOWNLOAD-FILE - Downloads file from specified url
Example - DOWNLOAD-FILE <URL> <OUT FILE>
